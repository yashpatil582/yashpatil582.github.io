# Yash Patil's Portfolio

A modern, interactive portfolio website showcasing my work in AI/ML, Data Science, and Software Engineering.

**Live Site:** [yashpatil582.github.io](https://yashpatil582.github.io)

## Features

- **Dark/Light Mode** - Toggle between themes with smooth transitions
- **Particle.js Background** - Interactive animated particle system
- **Typing Animation** - Dynamic role display in hero section
- **Project Filtering** - Filter projects by category (AI/ML, LLM, Data, Web)
- **Animated Skill Bars** - Visual representation of expertise
- **Interactive Timeline** - Scroll-animated work experience
- **3D Tilt Effect** - Project cards with perspective transform on hover
- **Custom Cursor** - Unique cursor design for desktop
- **Scroll Progress Indicator** - Track page scroll position
- **Smooth Scroll Animations** - AOS library integration
- **Responsive Design** - Mobile-first approach

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6+)
- Particles.js
- AOS (Animate on Scroll)
- Bootstrap Icons

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. Or deploy to GitHub Pages
